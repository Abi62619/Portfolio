2D Endless Runner

More Information

The game ‘Cosmic Mission’ game genre is an endless 2D runner where the player can collect coins, they have to try and get further and further each time this will be put on the leaderboard. The objective of the game will be to get back to earth however, because it's an endless runner this will never happen. The game progression will cause the game to get harder as the player survives.
